# 📱 Digital Well-Being Analyzer

## Description

Digital Well-Being Analyzer is a Machine Learning-based application that helps users understand and improve their digital lifestyle. The system analyzes user screen time, social media usage, sleep habits, notifications, and other digital behavior patterns to predict their level of digital well-being.

Using data analysis, visualization, and predictive modeling, the application provides personalized insights into how technology usage impacts productivity, focus, and overall well-being. The project includes data preprocessing, exploratory data analysis (EDA), machine learning model training, model evaluation, and an interactive Streamlit dashboard for real-time predictions.

### Key Features

* 📊 Analyze daily digital device usage patterns
* 📱 Monitor screen time and social media usage
* 🔔 Evaluate notification frequency and distractions
* 😴 Assess the relationship between sleep and device usage
* 🤖 Predict Digital Well-Being Status using Machine Learning
* 📈 Interactive visualizations and insights
* 🌐 User-friendly Streamlit web application

### Project Objective

The primary objective of this project is to promote healthier digital habits by providing users with data-driven insights and predictions about their digital well-being. It helps identify excessive screen time, poor sleep patterns, and digital distractions, enabling users to make informed decisions for a balanced and productive lifestyle.
